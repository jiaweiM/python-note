# PyInstaller 安装

Last updated: 2022-09-16, 12:52
****

## 安装

```bash
pip install pyinstaller
```

验证安装：

```bash
pyinstaller --version
```

## 安装的命令

完整安装将如下命令放到执行路径上。

|命令|功能|
|---|---|
|pyinstaller|打包程序的主命令|
|pyi-makespec|用来创建 spec 文件|
|pyi-archive_viewer|用来检查打包程序|
|pyi-bindepend|查看可执行文件的依赖项|
|pyi-grab_version|从 Windows 可执行文件中提取版本信息|
