# Streamlit 安装
2023-04-21

## 安装 conda

- 在 https://docs.conda.io/en/latest/miniconda.html 下载最新版本的 conda
- 安装并运行

## 新建 conda 环境

例如，创建一个 Python 3.9 的新环境：

```bash
conda create -n stenv python=3.9
```

## 激活 conda 环境

```bash
conda activate stenv
```

## 安装 Streamlit 库

```bash
pip install streamlit
```

## 启动 Streamlit 示例

激活 stenv 环境后，在命令中输入：

```bash
streamlit hello
```
